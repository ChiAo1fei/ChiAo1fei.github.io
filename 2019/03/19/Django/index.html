4. 查

   - filter(字段=值)

   - get(条件)  --  条件必须成立并且查询结果只有一个

   - exclude(条件)   --  将满足条件的信息过滤掉

   - count()  --  查询结果的个数

   - values()  --  将对象的内容以字典的形式取出来、json

   - order_by('字段')

     - 升序：order_by(字段)
     - 降序：order_by('-字段')

   - contains:包含  --  filter(字段__contains=值)（以下的用法和contains的用法格式一样）

   - startwith  --  以什么开头

   - endwith  --  以什么结尾

   - 大于、小于

     - 大于、大于等  --  gt,gte
     - 小于、小于等  --  lt,lte

   - aggregate:聚合  --  Min, Max, Count, Sum, Avg

   - F  --  对象两个属性字段，可以进行加减算法  --  filter(字段1__gt=F(字段2))

     

   - Q  --  用于  与或非

     

   - all()  --  queryset结果  查询的结果是一个集合

   - first()  --  取结果中的第一个对象

   - last()  --  去结果中的最后一个对象
